This project was developed during 2023/24 as my undergraduate dissertation project at the University of Exeter.
The classification model uses the ResNext-50 architecture, the paper for which can be found here.
The detection model uses the Faster R-CNN architecture, the paper for which can be found here.
The dataset used was the GRAZPEDWRI-DX dataset, which was comprised of 20,327 images of wrist fractures.
The dataset was anonymised prior to this project, but as part of pre-processing for classification,
the minority class (Not Fractured) was augmented to match the majority class's size.
The augmentation steps can be found under the augmentation.py and checkdatasetbalance.py in the GitHub repo.
